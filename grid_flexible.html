Fluid Behavior Using Phone and Tablet Classes

Notice that if you omit tablet, the default fluid behavior will kick in (your column will become full width). If you narrow you viewport to about tablet width (~480w-900w), you should see the first four columns exhibiting this below.

So the "take-away" is if you choose to use phone-X for more control, you will likely want to also use tablet-X. Using phone-X and tablet-X together will likely result in a more predictable transition that you're expecting.
